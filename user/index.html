使用者指南

欢迎使用 AutoPku —— 北京大学本科全自动解决方案。从课程通知获取到作业提交，全程由 Agent Team 托管。你只需用自然语言下达指令，剩下的交给 AI。

AutoPku 是什么？

AutoPku 是一套面向北京大学本科生的智能课程管理 Agent 系统。它通过模拟人类操作教学网（Course），自动完成通知同步、作业解析与解答、笔记生成等重复性学业任务。

核心理念：人对 AI 说一句话，AI 帮你跑完全流程。

💡 典型使用场景

"帮我同步这周的通知"、"写一下操作系统实验班的笔记"、"量子力学的第五次作业还没做，帮我搞定" —— 类似这样的自然语言指令，AutoPku 都能理解并执行。

三大核心任务

📢
同步课程通知
自动拉取教学网上的作业、公告、截止日期，按课程分类整理到本地目录，确保你不遗漏任何重要信息。
📝
完成作业
解析 PDF 作业题目 → AI 解答 → 渲染为 PDF → 用户确认 → 自动提交。全流程托管，你只需在最后点一次确认。
📚
撰写课程笔记
从课件提取核心内容，生成带 LaTeX 公式的 Markdown 笔记，支持数学推导与定理整理。

安全与隐私

⚠️ 作业提交前必须用户确认

AutoPku 的作业提交环节设计了强制人工确认机制。AI 完成解答并渲染 PDF 后，会暂停并提示你审阅内容。只有在你明确同意后，才会执行最终提交操作。这是为了防止未经审核的自动提交带来的学术风险。

🔒 本地优先架构

你的课程数据、登录凭据、生成的作业与笔记默认全部保存在本地。除非明确配置云同步，否则不会有任何学业数据离开你的设备。
