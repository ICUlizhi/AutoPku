# 测试：sync-notices — 多课程并行同步

## 场景描述
用户要求同步多门课程的通知。Agent 应并行处理各课程，正确解析 ANSI 颜色码，分别为每门课程生成通知摘要。

## 前置条件
- pku3b 已安装
- 已登录教学网

## 输入
- 用户意图: `"同步逻辑导论和哲学导论的通知"`
- mock 数据: fixtures/mock_assignments.txt

## 期望输出
1. test00/逻辑导论/ 和 test00/哲学导论/ 目录结构均完整
2. 每门课程的通知摘要.md 包含对应课程统计、待交作业、下载文件列表
3. 无 ANSI 颜色码残留

## 验证命令
python validate.py
