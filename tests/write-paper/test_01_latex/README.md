# 测试：write-paper — LaTeX 模式论文生成

## 场景描述
用户要求生成某课程的 LaTeX 格式论文。Agent 应正确获取论文要求、生成大纲与正文、
替换模板占位符，最终产出可编译的 `paper.tex`。

## 前置条件
- pku3b 已安装且已登录
- xelatex 可用（或至少 tex 语法正确）

## 输入
- 用户意图: `"给马原写课程论文，用 LaTeX 格式"`
- 模拟课程: `test00/马原/`
- 论文要求文件: `通知/论文要求.md`

## 期望输出
1. `test00/马原/论文/paper.tex` 存在
2. 模板占位符（`在此填写题目`、`在此填写姓名` 等）已被实际内容替换
3. 正文各章节 LaTeX 代码已插入到正确位置
4. 无残留未替换占位符
5. 文件语法层面可通过 xelatex 编译（或至少结构正确）

## 验证命令
```bash
python validate.py
```
