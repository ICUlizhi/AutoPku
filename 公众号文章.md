## 技术方案：AutoPku = pku3b + Claude Code

这套系统的核心是两个神器：

### 1. pku3b - 命令行教学网客户端

[pku3b](https://github.com/yang-er/pku3b) 是 @yang-er 开发的教学网命令行工具。它让你可以用命令行操作教学网：

```bash
# 查看所有作业
$ pku3b assignment list

# 下载附件
$ pku3b assignment download 12345

# 提交作业
$ pku3b assignment submit 12345 ./homework.pdf
```

再也不用打开浏览器、登录教学网、逐个课程点开看了。

### 2. Claude Code - AI 驱动的自动化引擎

[Claude Code](https://github.com/anthropics/claude-code) 是 Anthropic 推出的 AI 编程助手。它的特点是：

- 可以直接执行系统命令
- 可以读写文件、操作Git
- 可以理解复杂任务并自主完成
- 支持多 Agent 协作（Agent Team）

结合起来，我们可以让 Claude 自动：

1. 调用 `pku3b` 获取通知
2. 解析和分类通知
3. 创建文件夹、移动文件
4. 生成 Markdown 报告

## 手把手教程：从零搭建 AutoPku

### 第一步：安装 pku3b

**Windows**（PowerShell）：
```powershell
irm https://github.com/yang-er/pku3b/raw/main/install.ps1 | iex
```

**macOS/Linux**：
```bash
curl -fsSL https://github.com/yang-er/pku3b/raw/main/install.sh | sh
```

**配置登录**：
```bash
pku3b auth login
# 按提示输入学号和密码

# 测试
pku3b assignment list
```

看到作业列表？恭喜，第一步完成！

---

### 第二步：安装 Claude Code

```bash
npm install -g @anthropic-ai/claude-code
```

安装完成后，在项目目录运行：
```bash
claude
```

---

### 第三步：创建配置文件

在你的课程项目根目录创建 `skill.md`：

```markdown
# AutoPku 配置

## 课程列表

```yaml
courses:
  - name: "量子力学"
    folder: "./量子力学"
    keywords: ["量子", "简明量子力学"]
  
  - name: "马原"
    folder: "./马原"
    keywords: ["马原", "马克思主义基本原理"]
  
  # 添加你的课程...
```

## Agent 任务

你是 AutoPku Agent，负责：

1. 运行 `pku3b assignment list --json` 获取所有通知
2. 根据 keywords 将通知分配到对应课程
3. 在每个课程的 folder 下创建：
   - 作业/  - 存放作业要求
   - 通知/  - 存放一般通知  
   - 资料/  - 存放课件资料
   - 通知摘要.md - 课程任务清单
4. 生成 dashboard.md 总览报告
```

---

### 第四步：运行自动化

在 Claude Code 中加载配置：

```
claude > 读取 skill.md 并执行
```

Claude 会自动：
1. 调用 `pku3b` 获取数据
2. 创建课程文件夹
3. 生成通知摘要

---

### 第五步：设置定时任务（可选）

让系统自动每天运行：

**macOS/Linux** - crontab：
```bash
# 编辑定时任务
crontab -e

# 添加：每天早8点自动同步
0 8 * * * cd /path/to/your/courses && claude run skill.md
```

**Windows** - 任务计划程序：
1. 创建基本任务
2. 设置触发器：每天 8:00
3. 设置操作：启动程序 `claude`，参数 `run skill.md`

## 进阶玩法

### 1. 多 Agent 并行处理

如果你的课程很多，可以为每门课创建一个 Agent：

```
Team: course-notifiers
├── Agent: 马原-agent
├── Agent: 量子力学-agent
├── Agent: 毕设-agent
└── ...
```

Claude 会并行处理所有课程，速度更快。

### 2. 企业微信推送

配置 webhook，新作业自动推送到手机：

```yaml
notifications:
  webhook:
    url: "https://qyapi.weixin.qq.com/cgi-bin/webhook/..."
    events: ["新作业发布", "DDL提醒"]
```

### 3. 集成 Obsidian/Notion

将生成的 Markdown 报告同步到笔记软件：

```bash
# 软链接到 Obsidian 仓库
ln -s ./reports/dashboard.md ~/Obsidian/PKU/
```

## 常见问题

**Q: pku3b 登录失败？**
- 检查学号和密码是否正确
- 确认网络可以访问教学网
- 尝试 `pku3b auth logout` 后重新登录

**Q: Claude Code 无法执行命令？**
- 确保在 Claude Code 中允许了 Bash 权限
- macOS 用户可能需要给 Terminal "完全磁盘访问权限"

**Q: 课程匹配不准确？**
- 在配置中多添加几个关键词
- 先用 `pku3b assignment list` 查看原始课程名
