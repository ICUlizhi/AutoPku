一起共建 AutoPku

你不会写代码？没关系，让 AI 帮你写。

💡 什么是 Vibe Coding？

Vibe Coding 是一种全新的开发方式：你描述需求，AI 写代码。

你不需要记住语法，不需要精通编程，只需要清晰表达想法。AutoPku 本身就是用 Vibe Coding 构建的——现在，我们邀请你一起加入这场有趣的实验。

为什么小白也能贡献？

🤖
AI 写代码
你只需要说"帮我加个功能"，AI 就会自动写代码、改代码。你是指挥家，AI 是乐手。
📋
任务分级
GitHub Issues 有简单 / 中等 / 困难标签，新手从"good first issue"开始，循序渐进。
💬
全程有指导
社区维护者会帮你 review 代码，AI 会帮你修改。不用担心犯错，大家都在学习。
🎓
边做边学
在贡献中自然学会 Git、Markdown、Python——不是枯燥的上课，而是动手做中学。

共建六步流

从零到 Contributor，只需要跟着这六步走。每一步都有 AI 和社区陪着你。

1

先用起来

参考使用指南，把 AutoPku 跑起来，成为真正的用户。只有用过，才知道哪里可以更好。

2

创建分支

对 AI 说"帮我 Fork 仓库并创建一个 feature 分支"，AI 会一键搞定。你甚至不用理解 Git 的原理。

3

看 Issue 领任务

去 GitHub Issues 逛逛，找感兴趣的、标有 good first issue 的任务，留言"我想做这个"。

4

Vibe Coding

对 AI 说"帮我实现这个功能"——描述清楚你想要什么，AI 会写代码、改文件。你负责思考和把关。

5

测试 & 提 PR

让 AI 帮你跑测试，确认没问题后，对 AI 说"帮我提交并推送到远程分支"。然后在 GitHub 上发起 Pull Request。

6

合并 & 庆祝 🎉

维护者 review 后合并你的代码，你的名字会永久出现在 Contributors 列表中。恭喜你，正式成为开源贡献者！

你需要准备什么？

  • 一个 GitHub 账号（免费注册，一分钟搞定）
  • 一个 AI 编程助手（Claude Code / Cursor / Kimi 都可以，选一个你喜欢的）
  • 一颗愿意尝试的心 ❤️
⚠️ 重要提醒
  • 🚫 不要在代码中提交你的学号、密码等敏感信息
  • 💬 不确定的时候多问，社区很友好，没有 stupid question
  • ✏️ 哪怕只是改个错别字、补充一句话，也是宝贵的贡献
